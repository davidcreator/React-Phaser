# 📬 Contato e Suporte

## Objetivo
Criar canais claros para dúvidas, bugs e sugestões.

## Canais recomendados
- E-mail de suporte
- GitHub Issues
- Discord (opcional)
- Formulário simples

## Modelo pronto
- E-mail: suporte@seudominio.com
- GitHub: github.com/seuusuario/seu-jogo
- Discord: discord.gg/seuservidor
- Prazo de resposta: 48h úteis

## Template de resposta
"Recebemos seu relato. Vamos reproduzir o problema e retornar com atualização em até 48h úteis. Obrigado pelo feedback!"

## Boas práticas
- Linguagem simples e objetiva
- Pedir passos para reproduzir o bug
- Registrar tudo no backlog
