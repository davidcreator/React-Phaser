# 04 - Fisica

Modelos de fisica usando Arcade Physics do Phaser.

## O que voce vai praticar
- Gravidade
- Quicar (bounce)
- Colisao entre objetos
- Atrito/arrasto (drag)
- Forcas simples e interacao entre corpos

## Dica
Ajuste valores de gravidade, bounce e drag para sentir o comportamento no jogo.
