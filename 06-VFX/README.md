# 06 - VFX

Esta pasta reúne jogos e exemplos com **efeitos visuais**.

## Conteúdo visto nesta pasta
- Partículas
- Efeitos de impacto
- Screen shake
- Flash de dano e feedback visual
- Transições visuais entre cenas

## Objetivo pedagógico
Mostrar como VFX melhora legibilidade de ações e sensação de resposta do jogo.
