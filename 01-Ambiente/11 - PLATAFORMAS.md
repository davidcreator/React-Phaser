# 🌐 Plataformas de Publicação (React + Phaser)

## Onde publicar jogos web
- itch.io
- GitHub Pages
- Netlify
- Vercel

## Comparativo rápido
| Plataforma | Facilidade | Custo | Melhor para |
|---|---|---|---|
| itch.io | Alta | Grátis | Comunidade gamer e jogos indie |
| GitHub Pages | Alta | Grátis | Portfólio e projetos de estudo |
| Netlify | Alta | Grátis com limites | Deploy contínuo simples |
| Vercel | Alta | Grátis com limites | Projetos React modernos |

## ✅ Checklist antes de publicar
- `npm run build` sem erros
- README atualizado
- Licença definida
- Créditos de assets incluídos
- Controles explicados

## Dica prática
Para turmas iniciantes, publique primeiro em itch.io ou GitHub Pages.
