# 🕹️ Tipos de Jogos para Criar com React + Phaser

## Gêneros recomendados para começar
| Gênero | Complexidade | Ideal para |
|---|---|---|
| Pong/Arcade | Baixa | Primeira semana |
| Puzzle | Baixa/Média | Lógica e UI |
| Plataforma 2D | Média | Física e colisões |
| Top-down shooter | Média | Input e inimigos |
| Endless runner | Média | Spawn e progressão |

## Como escolher o tipo certo
1. Tempo disponível.
2. Nível da turma.
3. Objetivo didático (física, IA, UI, etc).

## 📅 Trilha sugerida
- Semana 1: Pong
- Semana 2: Breakout
- Semana 3: Plataforma simples
- Semana 4: Projeto autoral curto

## Regra de ouro
Escolha um gênero que permita entregar um MVP jogável em até 2 semanas.
