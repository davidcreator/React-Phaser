# 08 - bouncing-com-limitador

Modelo de script React + Phaser para a licao "08 - bouncing-com-limitador".

## Arquivos principais
- index.html
- css/styles.css (ou variante de estilo)
- js/script.js

## Categoria
- 04 - Fisica

## Como usar no projeto React + Phaser
1. Copie o arquivo js/script.js para seu projeto em src/scenes ou src/scripts.
2. Importe e registre a cena no config do Phaser.
3. Ajuste o scene key sugerido: 04-fisica-08-bouncing-com-limitador
4. Personalize regras, assets e HUD conforme seu jogo.
5. Rode npm run dev para validar.
