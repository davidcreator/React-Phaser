# 05 - Sprites

Esta pasta foca em jogos e exemplos que utilizam **sprites**.

## Conteúdo visto nesta pasta
- Uso de sprites e spritesheets
- Animações 2D (idle, run, hit, etc.)
- Organização de assets visuais
- Escala e consistência visual entre personagens e cenário

## Objetivo pedagógico
Ensinar integração de arte 2D ao gameplay, mantendo clareza visual e performance.
