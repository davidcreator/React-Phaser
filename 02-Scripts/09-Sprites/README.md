# 09 - Sprites

Modelos para uso de sprites em jogos 2D com Phaser.

## Projetos
- `01-AdventureSprites`: base de sprite com movimento simples
- `02-RPG`: personagem com animacao de caminhada, camera e obstaculos
- `03-Puzzle`: drag and drop de pecas em grade com validacao de puzzle
- `05-Luta`: combate 1x1 com ataque, dano e barra de vida

## O que voce vai praticar
- Criacao e uso de sprite
- Movimentacao de personagem
- Base para animacao de spritesheet
- Integracao com controle e fisica

## Dica
Substitua as texturas geradas por assets reais em `/public/assets`.
