# 💻 Requisitos de Sistema — React + Phaser

## Para desenvolver
### Mínimo
- Windows 10+ / Linux / macOS
- Node.js LTS 18+
- 8 GB RAM
- VS Code
- Git

### Recomendado
- Node.js LTS mais recente
- 16 GB RAM
- SSD
- Navegador atualizado

## Para jogar (build web)
### Mínimo
- Navegador moderno com WebGL
- 4 GB RAM

### Recomendado
- 8 GB RAM
- CPU intermediária recente
- Conexão estável para assets

## Comandos base
```bash
npm create vite@latest meu-jogo -- --template react
npm install
npm install phaser
npm run dev
npm run build
```
