## Como usar os modelos
1. Abra a pasta da categoria.
2. Escolha uma licao.
3. Copie `js/script.js` para seu projeto React + Phaser.
4. Registre a cena no config do Phaser.
5. Adapte regras, assets e UI para seu jogo.

## Observacao
Os modelos foram padronizados para React + Phaser e servem como base didatica para alunos.
