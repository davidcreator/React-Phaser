# 03 - Cenarios

Modelos para construcao de cenarios com camadas, troca de fundo e parallax.

## O que voce vai praticar
- Composicao visual
- Fundo por camadas
- Troca de cenario em runtime
- Efeito de parallax

## Dica
Use esses modelos junto com a pasta de Sprites para criar cenas mais completas.
