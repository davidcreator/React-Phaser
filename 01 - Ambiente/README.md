# 🚀 Guia de Ambiente — React + Phaser

Este material foi adaptado para a trilha de desenvolvimento de jogos web com **React + Phaser**.

## 🎯 Objetivo
- Configurar o ambiente de desenvolvimento.
- Padronizar a estrutura dos projetos da turma.
- Criar base sólida para jogos 2D no navegador.

## 🧰 Stack da trilha
- **React**: interface e organização da aplicação.
- **Phaser**: engine de jogo 2D.
- **Vite**: servidor de desenvolvimento e build.
- **Git + GitHub**: versionamento e colaboração.

## ⚡ Setup rápido
1. Instale o Node.js LTS.
2. Instale VS Code e Git.
3. Crie o projeto:
```bash
npm create vite@latest meu-jogo -- --template react
cd meu-jogo
npm install
npm install phaser
npm run dev
```

## 📁 Estrutura recomendada
```txt
src/
  scenes/
  objects/
  ui/
  PhaserGame.jsx
  App.jsx
```
