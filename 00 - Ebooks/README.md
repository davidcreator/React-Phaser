# 00 - Ebooks

Esta pasta reúne **livros essenciais** para base de programação de jogos.

## Conteúdo visto nesta pasta
- Lógica de programação
- Algoritmos
- Estruturas de decisão e repetição
- Pensamento computacional
- Organização de raciocínio para resolver problemas

## Objetivo pedagógico
Dar ao aluno uma base sólida antes de avançar para código de jogo em React + Phaser.

## Como estudar
1. Ler os materiais introdutórios de lógica.
2. Praticar exercícios simples de algoritmo.
3. Voltar aos livros sempre que travar em implementação.
