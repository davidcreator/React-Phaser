# 🔞 Classificação Etária para Jogos Web

Guia rápido para definir classificação indicativa do jogo.

## 🎯 Objetivo
- Proteger o público correto.
- Evitar conteúdo inadequado para a faixa etária.

## 📊 Referência rápida (Brasil)
| Faixa | Uso comum |
|---|---|
| Livre | Conteúdo casual e educativo |
| 10+ | Conflito leve |
| 12+ | Ação moderada |
| 14+ | Temas mais intensos |
| 16+ | Conteúdo maduro |
| 18+ | Conteúdo explícito |

## 🧭 Como decidir
1. Analise violência, linguagem e tema.
2. Verifique se há interação online.
3. Escolha a faixa mais segura.
4. Registre a decisão no documento do projeto.

## ✅ Checklist
- Tem violência explícita?
- Tem linguagem ofensiva?
- Tem sustos intensos?
- Tem compras internas?
- Tem chat entre jogadores?

## 📌 Boas práticas
- Seja claro na descrição do jogo.
- Atualize a classificação após mudanças no conteúdo.
